Fetching and persisting data

All data is fetched from Riot's Europe server - EUNE.

Data is fetched from the last week, only from the ranked queues.

If some data is corrupt and can't be parsed - it will be omitted in the final score.

The engine will persist calculated data for a week and then fetch straight from a MySQL DB, based on the called playerId. For presentation purposes - you can refresh the db.
